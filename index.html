Minha Jornada

HTML e CSS

O começo da minha jornada na programação se iniciou com HTML e CSS aprendendo passo por passo diretamente do zero já que eu tive pouco contato com computadores na infância, tudo parecia um grandissimo desafio mas isso era o que tornava mais interessante e glorioso, cada passo foi uma conquista e nunca se limitando a aprender, tendo em mente que a tecnologia é uma área constantemente em evolução precisa sair da zona de conforto o tempo inteiro.

JavaScript e PHP

Como segunda fase da minha jornada, aqui eu cheguei bem mais preparado ja tirando de letra HTML e CSS mas não imaginava o desafio que estava por vir, o JavaScript foi primeiramente um susto por completo cheio das suas lógicas e códigos novos para desbravar, apesar da insegurança, aquelas linhas de códigos lógicos nunca deixaram de me cativar, pós isso veio o PHP e como ja estava freco na memória os caminhos a seguir foi uma estrada menos longa mas muito desafiadora.

PHPBB e MySQL

Após o PHP o próximo passo foi ir em busca de aprendizado no PHPBB que é um sistema gerenciador de fóruns para a Internet construído através de scripts em PHP, após o PHPBB cheguou o tão temido Banco de Dados, o MySQL, cada avanço basta muita atenção e persistência, o MySQL é extramamente fascinante e tenho muito interesse de aperfeiçoar cada vez mais nesse nicho.
